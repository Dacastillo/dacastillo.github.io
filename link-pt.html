Links de Interesse

Recursos úteis para pesquisa em física computacional, computação quântica e análise bibliométrica.

APIs Bibliométricas

  • OpenAlex
    Banco de dados aberto de trabalhos científicos, autores e conceitos. API gratuita.
  • Semantic Scholar API
    API de busca de artigos com análise de citações e relevância.
  • CrossRef API
    Metadados de publicações acadêmicas com DOI.
  • ORCID
    Identificador único para pesquisadores. Meu ORCID: 0000-0002-9556-4831

Ferramentas de Simulação

  • LAMMPS
    Simulador de Dinâmica Molecular. Código aberto para dinâmica molecular.
  • GPAW
    Código DFT baseado no método projector-augmented wave.
  • Quantum ESPRESSO
    Suite integrada de códigos para simulação de estrutura eletrônica.
  • ORCA
    Software de química quântica ab initio, DFT e semiempírico.
  • Qiskit
    SDK de computação quântica da IBM. Baseado em Python.

Machine Learning para Ciência

  • PyTorch
    Framework de deep learning para Python.
  • scikit-learn
    Ferramentas de machine learning para Python.
  • ASE (Atomic Simulation Environment)
    Biblioteca Python para ciência de materiais computacional.
