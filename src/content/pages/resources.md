## Open Code Repositories

Here are some of my key open-source projects available on GitHub:

### MLIP Training Pipeline

A comprehensive workflow for training machine learning interatomic potentials using active learning.

- **Repository:** [github.com/Dacastillo/mlip-pipeline](https://github.com/Dacastillo)
- **Technologies:** Python, GPAW, AMP, LAMMPS
- **Status:** Active development

### Quantum Chemistry Tools

Collection of tools for quantum computing applications in chemistry, including VQE implementations and molecular integrals.

- **Repository:** [github.com/Dacastillo/quantum-chem](https://github.com/Dacastillo)
- **Technologies:** Python, Qiskit, PennyLane
- **Status:** Active development

### Bibliometric Analysis Toolkit

Python scripts and Jupyter notebooks for analyzing scientific literature trends.

- **Repository:** [github.com/Dacastillo/bibliometrics](https://github.com/Dacastillo)
- **Technologies:** Python, pandas, bibliometrix, matplotlib
- **Status:** Completed

## Datasets & Tools

### hBN Training Data

Dataset of hexagonal boron nitride structures and energies for MLIP training.

- **Format:** XYZ + energy/force labels
- **Size:** ~10,000 configurations
- **License:** CC BY 4.0

### Diamond Nanoparticle Structures

Crystallographic data for diamond nanoparticles under various pressure conditions.

- **Format:** CIF files
- **Includes:** DFT calculation inputs/outputs
- **License:** CC BY 4.0
